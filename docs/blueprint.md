# **App Name**: ServiceFlow Pro

## Core Features:

- Client Profile Management: Create, update, and manage client profiles with detailed contact information and service history.
- Service Request Creation: Allow clients or admins to create service requests detailing the issue, urgency, and required services.
- Task Assignment and Tracking: Assign tasks to team members, set deadlines, and track task progress to ensure timely service delivery.
- Service Status Updates: Update service statuses (e.g., 'Pending', 'In Progress', 'Completed') to keep clients informed of the progress.
- Admin Dashboard with Chart Section: Dedicated admin area with detailed charts summarizing services completed, pending requests, and team member performance for 'shairahaman.csc@gmail.com'.
- Techician Dashboard with Chart Section: Dedicated tech area with detailed charts summarizing services completed, pending requests, and personal task performance for 'sskingshaikpro@gmail.com'.
- AI-Powered Request Summarization: Use AI to summarize the user's request to determine the overall sentiment.

## Style Guidelines:

- Primary color: Professional blue (#3498db) to convey trust and reliability.
- Background color: Light gray (#ecf0f1), providing a clean and modern backdrop.
- Accent color: A darker shade of blue (#2980b9) used for key actions and highlights to guide user attention.
- Body and headline font: 'Inter', a grotesque-style sans-serif, will be used to ensure a modern and neutral feel.
- Use clean, minimalist icons to represent different service types and statuses.
- A clean, intuitive Neumorphism layout with clear visual hierarchy to allow users to navigate the application effectively.
- Subtle transitions and animations to provide feedback on user interactions and enhance the overall experience.
